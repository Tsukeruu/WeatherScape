# Security Policy


## Reporting a Vulnerability

- **If a vulnerability were found**
  - *Please kindly do not open a public issue as this adds fuel to the fire even more*
  - Contact the maintainer *privately* with the details or open a private security advisory on GitHub
  
  ### How to explain the vulnerability
  - **After getting in touch, please provide the following details:**
    - *A clear concise description of the vulnerability*
    - *Where you found it*
    - *What caused it*
    - *Potential impact if known*
